## Installation

### **Required Environments:**
- PHP >= 8.0
- Mysql
- Make a new database to set in ENV file

### **To install and run project please execute the below commands:**

- composer install
- php artisan key:generate
- php artisan migrate
- php artisan db:seed
- php artisan test `// to test the project if needed`
